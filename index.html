Ecuaciones Diferenciales

Una aplicación web para la resolución y enseñanza de los métodos derivados de las ecuaciones diferenciales.

Acerca

La creación del proyecto

La razón de esta aplicación web es para simplificar la resolución y el aprendizaje de los métodos que tienen las ecuaciones diferenciales. El programa busca ser una herramienta práctica para estudiantes y profesionales, permitiéndoles obtener la respuesta a problemas de forma rápida y eficaz, mientras a su vez comprenden el procedimiento paso a paso. Creemos que mediante nuestra estructura 4 secciones por página el usuario podrá aprender completamente el método perteneciente a la unidad de ecuaciones diferenciales:

  • Escrito - redacción del tema mostrando fórmulas y conceptos clave para los problemas.

  • Gráfico - presentación de imágenes sobre problemas del tema resueltos.

  • Audiovisual - grabación de un problema siendo resuelto y explicado por el profesor.

  • Interactivo - problema con entrada de datos para la respuesta y correción errores.

Objetivos de la aplicación:

  • De fácil uso para el usuario

  • Completo entendimiento del tema

  • Comprensión de los problemas aplicados

  • Comprensión de los problemas calculados
