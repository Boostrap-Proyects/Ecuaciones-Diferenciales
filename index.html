Acerca

La creación del proyecto

La razón de crear esta aplicación web es para simplificar la resolución y el aprendizaje de los métodos que tienen las ecuaciones diferenciales. El programa busca proporcionar herramientas prácticas y accesibles para estudiantes y profesionales, permitiéndoles comprender y resolver problemas de manera eficiente. Mediante la estructura de página que tiene cada método se pretende que el tema quede comprendido completamente para el usuario, es por eso que se optó por 3 secciones:

  • Redacción de un resumen del tema presentando los conceptos y fórmulas principales, además de un video del profesor resolviendo un problema mientras lo va explicando.
  • Imagen de un problema resuelto hecho en libreta
  • Un problema interactivo con entrada de datos para que el usuario lo resuelva, calificando sus respuestas por cada paso.

Objetivos:

  • Ser de fácil uso para el usuario

  • Obtener entendimiento del tema

  • Comprensión de los problemas de aplicación

  • Comprensión de la solución del problema
